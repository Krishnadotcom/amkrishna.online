AutopentestX

"A Python and Django-based tool designed to automate routine penetration testing tasks through a simple GUI. It integrates modules for SSL/TLS validation, security headers analysis, port scanning, subdomain enumeration, and directory brute-forcing. Unlike individual command-line tools, AutoPentestX provides a unified workflow with automated reporting, reducing manual effort and improving consistency."
